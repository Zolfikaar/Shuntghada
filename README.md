# شنتغده؟ (Shuntghada) 🍽️

**شنتغده؟** هو تطبيق هاتف ذكي (Local-First) مبني باستخدام إطار عمل **.NET MAUI**، مصمم خصيصاً لحل واحدة من المعضلات اليومية في البيت العربي: *"شنتغده اليوم؟"* و*"شنتعشى؟"*.

على عكس تطبيقات الوصفات التقليدية التي تكتفي بسرد المكونات، يركز تطبيق "شنتغده" على كونها **أداة ذكية لإدارة القرار اليومي** عبر نظام مقترحات تفاعلي يمنع حيرة الاختيار ويراعي عدم تكرار الأكلات في فترات متقاربة.

---

## ✨ الميزات الرئيسية (Core Features - MVP)

* 🎲 **محرك الاقتراح الذكي (Suggestion Engine):** بضغطة زر واحدة، يقترح التطبيق أكلة عشوائية بناءً على تصنيفات المطبخ الشعبي.
* 🛡️ **نظام منع التكرار (Cool-down System):** بمجرد اعتماد أكلة اليوم، يتم تشفيرها وحفظها في السجل المحلي لمنع ظهورها في اقتراحات الأيام الـ 7 القادمة.
* 📖 **دليل الوصفات السريع:** عرض نبذة مسبقة عن الأكلة، الاسم البديل (حسب تنوع المناطق)، وخطوات الإعداد بضغطة زر وبدون مغادرة الصفحة.
* 📱 **معمارية Local-First:** يعتمد التطبيق كلياً على قاعدة بيانات محلية خفيفة (`SQLite`)، مما يضمن سرعة فائقة، استهلاكاً منعدماً للبطارية، وتكلفة تشغيل صفريّة بدون الحاجة لـ Backend معقد في مرحلة الإطلاق.

---

## 🛠️ البنية التقنية (Tech Stack)

* **Framework:** .NET MAUI (Targeting .NET 9.0-android)
* **Database / ORM:** SQLite-net-PCL & SQLitePCLRaw.bundle_green
* **Architecture:** Decoupled Model-Driven Architecture (System Builder Mindset)

---

## 🗺️ خطة إدارة الفروع (Branching Strategy)

يتبع المشروع استراتيجية إدارة فروع صارمة مستلهمة من مفاهيم كتاب *The Pragmatic Programmer* لضمان استقلالية الخصائص واستقرار فرع الإنتاج الرئيسي:

1. `feature/01-database-setup` - بناء الكيانات (Entities) وتأسيس قاعدة البيانات والـ Seed Data.
2. `feature/02-suggestion-engine` - بناء خوارزمية الاقتراح والـ History Logic.
3. `feature/03-basic-ui` - ربط الـ Backend بالواجهات الأساسية (XAML).
4. `feature/04-ui-ux-enhancements` - إدخال الموارد المرئية (الصور)، الأنيميشن، وتثبيت الأبعاد.
5. `feature/05-smart-filters` - تفعيل الفلاتر المتقدمة (القسم، الصعوبة، نوع المطبخ).

## 🚀 التشغيل والتطوير المحلي

### المتطلبات الأساسية
* .NET 9.0 SDK أو أحدث.
* بيئة عمل تدعم MAUI (مثل VS Code مع الـ MAUI Extension أو Visual Studio).
* محاكي أندرويد (Android Emulator).

### خطوات التشغيل
1. قم بعمل `Clone` للمستودع.
2. افتح الـ Terminal في المجلد الرئيسي وشغل الأمر:
```bash
dotnet restore
```

لتشغيل التطبيق مباشرة على المحاكي:

```Bash
dotnet build -t:Run -f net9.0-android
```

لاستخراج نسخة الـ APK الـ Release الخاصة بجهازك الشخصي:

```Bash
   dotnet publish -f net9.0-android -c Release /p:AndroidPackageFormat=apk /p:AndroidEnableSignApk=false /p:CheckEolTargetFramework=false
```
